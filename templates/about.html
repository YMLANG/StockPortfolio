How to use Niulio!


Running the application

1. Ensure the requirements.txt is present.
2. Ensure python3.6 as a default for python3 "alias python3=python3.6"
3. Install all python requirements "pip3 install -r requirements.txt"
4. (Optional) Do "chmod +x run.py" to simplify the running method.
5. Now run the service! "python3 run.py" or "./run.py" (if you had done the step 4)
